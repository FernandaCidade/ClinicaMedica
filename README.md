
# Clínica Médica

Este repositório contém o código-fonte de um sistema de controle de Clínica Médica.

### Ferramentas Utilizadas

 - SSMS 
 - SQL Server

Abra o arquivo no SSMS. Selecione o comando para usar o BD clinica_medica e clique em "Executar"
